# DevOps Automation Toolkit
Collection of GitHub Actions workflows, Terraform snippets, Helm charts, and Kubernetes manifests to accelerate CI/CD and infra automation.

![Banner](/mnt/data/A_banner_in_digital_graphic_design_features_Debasn.png)

## Contents
- `.github/workflows/` - reusable CI/CD examples
- `terraform/` - sample infra (S3, ECR)
- `k8s/` - sample k8s manifests & Helm chart
- `docs/` - best practices

## Quick wins
- Use the included `actions/deploy.yml` as a drop-in for your services.
